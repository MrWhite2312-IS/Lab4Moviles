# FrontEnd

Aplicación [Expo](https://expo.dev) con [file-based routing](https://docs.expo.dev/router/introduction).

## Empezar

1. Instalar dependencias

   ```bash
   npm install
   ```

2. Iniciar la app

   ```bash
   npx expo start
   ```

El código de las pantallas vive en `src/app`. Edita `src/app/index.tsx` para empezar.
